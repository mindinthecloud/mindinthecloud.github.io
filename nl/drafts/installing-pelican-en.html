Installing Pelican

2016/05/06

Just did a clean install of Pelican on a Linux Mint workstation. I’ll try to document all the steps up to a working site in this article.

rick@ender ~/Documents/ $ mkdir blog
rick@ender ~/Documents/ $ virtualenv blog
rick@ender ~/Documents/ $ cd blog/
rick@ender ~/Documents/ $ source bin/activate
(blog)rick@ender ~/Documents/blog $ pip install pelican
(blog)rick@ender ~/Documents/blog $ pip install Markdown
(blog)rick@ender ~/Documents/blog $ pip install Typogrify
(blog)rick@ender ~/Documents/blog $ pelican-quickstart
(blog)rick@ender ~/Documents/blog $ git clone --recursive https://github.com/getpelican/pelican-plugins
(blog)rick@ender ~/Documents/blog $ git clone --recursive https://github.com/getpelican/pelican-themes ./pelican-themes

You now have a basic Pelican install to work with. The rest is just a matter of choosing your hosting option and read the fine manuals on getpelican.com.

That’s all folks!
